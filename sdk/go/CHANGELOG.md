## [v0.4.0] - 2026-01-24

### Highlights

- Add namespace support, file loaders, and YAML parsing for agent definitions

### Added

- `Namespace` field on Agent struct for organizing agents into subdirectories
- `AllowedTools` field on Agent struct for tools that execute without confirmation
- `WithNamespace()` builder method on Agent
- `QualifiedName()` method returning `namespace/name` format
- `ParseQualifiedName()` function to split qualified agent names
- `LoadAgentFromFile()` to load Agent from markdown with YAML frontmatter
- `ParseAgentMarkdown()` to parse Agent from markdown bytes
- `LoadAgentsFromDir()` with recursive scanning and namespace derivation from subdirectory
- `LoadAgentsFromDirFlat()` for non-recursive loading (backward compatibility)
- `LoadTeamFromFile()` to load Team from JSON
- `LoadDeploymentFromFile()` to load Deployment from JSON
- YAML struct tags on all Agent fields
- `gopkg.in/yaml.v3` dependency for frontmatter parsing
